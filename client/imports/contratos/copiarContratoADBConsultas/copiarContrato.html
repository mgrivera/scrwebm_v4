Copiar un contrato a la base de datos de consultas
({{ processProgress.current }}/{{ processProgress.max }} - {{ processProgress.progress }})
({{ processProgress.message }})

Haga un click en el botón para copiar todos los contratos a la base de datos de consultas.

Esta función permite al usuario reiniciar el proceso de copia de contratos a la base de datos de consultas.

El resultado de ejecutar este proceso es que la próxima vez que se efectúe este proceso de copia, todos los contratos registrados serán copiadas nuevamente.

Este proceso puede ser útil cuando el usuario desee asegurarse que todos los contratos, en la base de datos de consultas, sean refrescados (actualizados); es decir, que la copia de estos contratos se efectúe nuevamente para asegurarnos que sus valores sean actualizados (y correctos).

Esta función revisa los contratos registrados para todas la empresas y copia a la base de datos de consultas los que no se han copiado antes, o se han copiado pero han sido modificados de alguna manera.

La base de datos de consultas permite tener un programa separado, desde el cual el usuario puede ejecutar consultas a los datos registrados.

Una vez copiado un contrato a la base de datos de consultas, el usuario puede iniciar la aplicación de consultas y efectuar consultas al contrato.

Los contratos pueden ser copiados muchas veces. Esta función puede copiar un mismo contrato muchas veces, si el usuario lo modifica luego de que ha sido copiado. De esta forma, la base de datos de consultas se mantiene actualizada, a medida que los contratos van siendo modificados en el tiempo.
