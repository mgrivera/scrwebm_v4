Permitimos enviar un Email desde el programa, para revisar si la configuración de este módulo es correcta.

El correo que se intenta enviar, tiene estas características:

  • to: smr.software@outlook.com
  • from: smr.software@gmail.com
  • cc: smr.software@gmail.com

Ud. debe hacer un click en el botón que se muestra más abajo y esperar un rato; luego revisar las direcciones de correo indicadas a ver si el correo ha llegado.

Nota: también se puede revisar la consola del server para el programa; si el Email está allí, es que ni siquiera se indicó una configuración en la veriable: mail_url, en la configuración (settings.json) en el programa.
